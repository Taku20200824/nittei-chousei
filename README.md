# まるばつ日程調整 ○✕

[調整さん](https://chouseisan.com/) 風の日程調整ツール。出欠の選択肢を **○ と ✕ の2つだけ** にしたシンプル版です。

**▶ 使う: https://taku20200824.github.io/nittei-chousei/**

## 特長
- タイトル・メモ・候補日（1行に1つ）でイベントを作成
- 参加者は名前＋各候補に **○ / ✕** で回答
- 全員の回答を表で集計、**○の合計**と**最有力候補**を自動ハイライト
- 回答の編集・削除に対応
- **サーバー不要**: 状態はURLに丸ごと埋め込まれ、リンクを送れば相手も同じ表を見て回答できます（端末内にも自動保存）

## 使い方
1. イベントを作成 → 「🔗 共有リンクをコピー」でリンクを相手に送る
2. 相手はリンクを開いて名前と○/✕を入力 → 登録すると**リンクが更新**される
3. 更新後のリンクを共有して回答を集めていく

## 制限
バックエンドを持たないため、複数人が**同時に別々に編集した内容は自動マージされません**。最新のリンクを回覧する運用（順番に回答 → 更新リンクを次の人へ）を想定しています。

依存ライブラリなし。`index.html` を開くだけで動きます。
